# Prompt IQ Test

[English](README.md)

Prompt IQ Test 是一个本地优先的桌面端应用，用来测评一个人使用 `LLM / AI Agent` 的能力。

它不只是判断用户会不会“写 Prompt”，而是判断用户能不能清晰定义目标、把模糊需求结构化、补足背景和约束、在多轮协作中持续纠偏，并最终把 AI 驱动到高质量结果。

## 当前状态

当前仓库仍处于规划阶段。

- 产品范围已经明确
- 核心交互模式已经明确
- 对外仓库 README 已完成

## 项目目标

Prompt IQ Test 主要回答两个问题：

1. 普通用户到底会不会用 AI？
2. 进阶用户能不能高质量地驱动 LLM / Agent？

`v1` 只面向个人用户。招聘、培训、企业评估等场景属于后续方向，不在首版范围内。

## 计划中的功能

### 测评模式

- `趣味速测`：更短、更轻、更偏参考值
- `全面测评`：覆盖更广、误判更少、诊断更稳
- `静态测评`：用户针对给定任务提交 Prompt
- `动态测评`：用户与执行 AI 多轮协作
- `题库抽样`：从本地题库抽题
- `AI 出题`：根据用户限定词或随机策略实时出题

### 静态测评

- `只评 Prompt`：只评估 Prompt 本身，不实际执行
- `执行后再评`：先调用执行模型，再结合执行结果评分

### 动态测评

- 用户在主聊天窗口中与执行 AI 交流
- 测评 AI 在侧边栏持续打分
- 会显示趋势和当前估分
- 结束后输出最终报告

### 首版核心评分关注点

首版会围绕固定核心 Rubric 打分，重点包括：

- 需求表达是否清晰
- 输出格式是否明确
- 能否把模糊问题变成结构化任务
- 背景信息和约束是否充分
- 能否主动发现并补齐缺失条件
- 是否具备迭代纠偏与结果验收意识

## 隐私边界

Prompt IQ Test 按“本地优先”原则设计。

- 程序不会把数据上传到作者服务器
- 程序永久免费开源
- 实际费用来自用户自己调用 AI 接口的 Token 消耗
- 用户发送给自己所配置 AI 服务商的内容，仍可能被该服务商看到
- 作者不对使用本软件造成的后果承担责任
